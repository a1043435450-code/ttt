# معمارية نظام صقر الرحبة

## مكونات النظام

### 1. Backend (Node.js + Express)
- REST API
- إدارة قاعدة البيانات
- منطق المحاسبة
- التكامل مع ZATCA
- المصادقة والصلاحيات

### 2. Frontend (React)
- واجهة مستخدم RTL
- لوحة تحكم
- إدارة الفواتير والمبيعات
- التقارير

### 3. Desktop (Electron)
- تطبيق مستقل على Windows
- قاعدة بيانات محلية
- عمل بدون إنترنت
- نظام الترخيص

### 4. Database (SQLite/PostgreSQL)
- شجرة حسابات
- القيود المحاسبية
- الفواتير والمبيعات
- المخزون
- المستخدمين

## تدفق البيانات

```
Desktop (Electron) 
    ↓
Backend API (Express)
    ↓
Database (SQLite)
    ↓
ZATCA Integration
```

## أمان النظام

- مصادقة JWT
- تشفير كلمات المرور
- صلاحيات حسب الدور
- سجل نشاط شامل
- توقيع رقمي للفواتير
